<h4>Damn Vulnerable Linux (DVL) Strychnine+E605 (1.4):</h4>
<p>Added more tools. Now Reverse Code Engineering tools is 99%, added Truecrypt, Eclipse IDE for Java and C++, added Mono for .NET vulnerability. Rearranged the menu, minor bug fixes ( :grin: ). We close tool addition with this and focus on bug fix and training material only from now on.</p>
<ul>
<li>[Application Development] Add Motor IDE</li>
<li>[Application Development] Update HLA to 1.98 and StdLib to 2.3</li>
<li>[Application Development] Add LogWatch</li>
<li>[DVL Core] Add XEN</li>
<li>[Reverse Code Engineering] Add Insight GDB Debugger</li>
<li>[Tutorials] Add CPU Sim - An Interactive Java-based CPU Simulator</li>
<li>[Reverse Code Engineering] Add JAD Java Decompiler</li>
<li>[Tools] Add VLC Media Player</li>
<li>[Documentation] Add TeTex</li>
<li>[Documentation] Add JabRef</li>
<li>[Application Development] Add Kile</li>
<li>[Documentation] Add kDissert Mindmapper</li>
<li>[Peneration Testing] Add JBroFuzz</li>
<li>[Application Development] Add WebScarab</li>
<li>[Peneration Testing] Add CAL9000</li>
<li>[Reverse Code Engineering] Add KDBG</li>
<li>[Application Development] Add xchm</li>
<li>[DVL Core] Add gtk libs</li>
<li>[Tools] Add xvidcap</li>
<li>[Tools] Add AcroRead</li>
<li>[Tools] Add Scite </li>
</ul>
<p>DVL 1.4 final is ready to go and is uploaded at the moment. We hit the 1.6 GB size, including all necessary to train software development, IT security and Reverse Code Engineering. During the next time the mirrors will be informed. After this we post the links. As well we do a short intro video to show all features and on how to use DVL.</p>
